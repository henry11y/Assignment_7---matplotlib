# Assignment_7---matplotlib


## Purpose
The purpose of this project is to use Python and Matplotlib to create data visualizations from real datasets. This assignment focuses on identifying trends and comparing values through graphs.

## Datasets Used
1. Fisher's Iris dataset from sklearn
2. loan_data.csv

## What the Project Includes
This project contains:
- 3 visualizations for the Iris dataset
- 3 visualizations for the loan dataset
- a short analysis paragraph about the loan data

## Visualizations
### Iris dataset
- Sepal Length vs Sepal Width by Species
- Petal Length vs Petal Width by Species
- Petal Length Distribution by Species

These graphs help show differences between iris species based on physical traits.

### Loan dataset
- Distribution of Income
- Income vs Credit Score
- Loan Approval Distribution

These graphs help show the spread of income, the relationship between income and credit score, and the number of approved versus denied loans.

## Implementation
The project was completed in a Jupyter Notebook using:
- pandas
- matplotlib
- sklearn.datasets

The Iris dataset was loaded from sklearn and converted into a pandas DataFrame.  
The loan dataset was loaded from a CSV file using pandas.  
Matplotlib was used to create scatter plots, a histogram, a boxplot, and a bar chart.


## Limitations
The loan dataset only includes a few variables, so the analysis is limited to income, credit score, gender, and loan approval. The visualizations show patterns, but they do not prove cause and effect.
